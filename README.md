# Data Cleaning Project

## Project Overview

The objective of this project is to clean and preprocess a real-world Airbnb dataset.

Data cleaning is a crucial step in the data analysis process because inaccurate, incomplete, and inconsistent data can negatively impact analysis and decision-making.

This project demonstrates various data cleaning techniques such as handling missing values, removing duplicates, detecting outliers, and correcting data formats.

## Dataset

Airbnb New York City 2019 Dataset

Number of Records: 48,895

Number of Features: 16

## Data Cleaning Steps

### Missing Value Handling

- Filled missing values in name column
- Filled missing values in host_name column
- Filled missing values in reviews_per_month column
- Handled missing values in last_review column

### Duplicate Removal

- Identified duplicate records
- Removed duplicate entries

### Outlier Detection

- Visualized outliers using boxplots
- Removed extreme price outliers using IQR method

### Data Type Correction

- Converted date columns into datetime format
